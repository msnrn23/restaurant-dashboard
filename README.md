# Restaurant Revenue Analytics Dashboard

A fully client-side analytics dashboard for restaurant revenue, customer, and marketing data.

**Live demo:** `https://YOUR_USERNAME.github.io/restaurant-dashboard/`

## Features
- **6 KPI cards** — total revenue, avg revenue, customers, marketing, reviews, promo rate
- **Multi-dropdown filters** — cuisine, promotions, revenue band, customers, marketing, reviews, menu price
- **5 analytical tabs** — Overview · Trends · Relationships · Cuisine Deep-dive · Data Table
- **Relationship views** — interactive scatter plot with axis selectors, correlation matrix, binned movement charts
- **Export** — filtered data as CSV
